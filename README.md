# OYO-Hotel-Booking-Revenue-Analysis

This project analyzes OYO hotel performance data to monitor revenue, occupancy, pricing efficiency, and realization metrics across cities, properties, and time periods using Power BI.
The dashboard enables stakeholders to track key hospitality KPIs and identify trends impacting overall business performance.
